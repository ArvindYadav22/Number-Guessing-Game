#  Number Guessing Game

A fun and interactive number guessing game built with vanilla JavaScript, HTML, and CSS. Test your intuition and try to guess the secret number between 1 and 100 in just 10 attempts!

##  How to Play

1. The game randomly selects a number between 1 and 100
2. Enter your guess in the input field
3. Click "Submit" to check your guess
4. You'll receive feedback:
   -  **Too low!** - Your guess is lower than the secret number
   -  **Too high!** - Your guess is higher than the secret number
   -  **Correct!** - You've guessed the number!
5. You have **10 attempts** to find the correct number
6. Track your previous guesses and remaining attempts
7. Start a new game anytime with the "Start New Game" button

##  Features

- **Interactive Gameplay** - Real-time feedback on each guess
- **Attempt Tracking** - See all your previous guesses and remaining attempts
- **Input Validation** - Ensures valid numbers between 1-100
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Dark Theme UI** - Modern, eye-friendly dark mode interface
- **Visual Feedback** - Emoji indicators and color-coded messages
- **Restart Functionality** - Easy game reset with one click

##  Technologies Used

- **HTML5** - Structure and semantic markup
- **CSS3** - Styling with modern design patterns
- **JavaScript (ES6)** - Game logic and interactivity

##  Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start guessing!

No build process or dependencies required - just open and play!
